LUAU for Roblox Game Developers

What is LUAU?

LUAU is used in Roblox Studio. Roblox Studio is a game engine made by the company ROBLOX and Roblox's game engine allows players to create and script their own games using the Syntax of LUAU. LUAU is a scripting language which allows for fast compiling and is very user friendly; it also allows for game development.



Syntax/Keyword Definition

function

 A group of code that can take input and be executed again and again

local

 Used to define if a variable is only usable in one place

if

 Used to define the start of a condition

then

 Used to define the end of a condition

while

 If the condition is met it will keep repeating
